      🔋 Athan not playing when the tab is in the background? Tips for reliability
      • Keep the window visible (even small, at the side) rather than minimized — browsers suspend minimized windows most aggressively.
      • Chrome / Edge: open Settings → Performance and add this site under “Always keep these sites active” so Memory Saver never puts the tab to sleep.
      • Play the Quran (the Quran tab) — a tab that is actively playing audio is kept awake by the browser, so listening to the Quran also keeps the athan schedule running.
      • Install it as an app (browser menu → “Install” / “Add to Home Screen”) — an app window is treated better than a background tab.
      • Keep the computer awake: a sleeping computer stops everything. On a Mac: System Settings → Lock Screen → prevent sleeping; on Windows: Power settings.
      • If the tab was suspended anyway, the app catches up when it wakes: recently missed athans (within 10 minutes) still play, older ones are listed in the Activity tab.

        Install the Desktop App

        This website works everywhere with nothing to install — but for a permanent, always-on athan (menu-bar icon, auto-start when the computer turns on), you can install the desktop application. Pick your system below; each link opens the exact step-by-step instructions on GitHub.

        • 🌙 macOS — Menu-Bar App with Auto-StartRecommended. The green crescent in the menu bar, starts at every login, keeps the Mac awake. ↗
        • MacBook M1 / M2 / M3 (Apple Silicon)Terminal setup from scratch — Homebrew, Python, VLC (arm64 warning included). ↗
        • 🖥 iMac / MacBook IntelTerminal setup for Intel Macs. ↗
        • 🪟 Windows 10 / 11Python + VLC setup in PowerShell. ↗
        • 🐧 LinuxRun as a systemd user service. ↗
        • ⚙️ macOS headless (no menu-bar icon)Background-only variant via launchd. ↗
        • ✅ Verify your installationTest that audio, prayer times and scheduling all work. ↗

        📱 On phones there is nothing to install: open this site and use your browser's “Add to Home Screen” / “Install” — it becomes an app with its own icon.
