      🔋 Athan not playing when the tab is in the background? Tips for reliability
      • Keep the window visible (even small, at the side) rather than minimized — browsers suspend minimized windows most aggressively.
      • Chrome / Edge: open Settings → Performance and add this site under “Always keep these sites active” so Memory Saver never puts the tab to sleep.
      • Play the Quran podcast (the 🎙 playlist below) — a tab that is actively playing audio is kept awake by the browser, so listening to the Quran also keeps the athan schedule running.
      • Install it as an app (browser menu → “Install” / “Add to Home Screen”) — an app window is treated better than a background tab.
      • Keep the computer awake: a sleeping computer stops everything. On a Mac: System Settings → Lock Screen → prevent sleeping; on Windows: Power settings.
      • If the tab was suspended anyway, the app catches up when it wakes: recently missed athans (within 10 minutes) still play, older ones are listed in the Activity log.
